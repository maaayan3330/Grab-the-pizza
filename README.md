# Grab the Pizza
A fast-paced interactive browser game built with React and Vite.  
Test your reflexes by clicking the correct pizza box before it disappears — the faster you react, the higher your score.

## ✨ Features
- Dynamic pizza box system with randomized behavior
- Real-time score tracking
- Countdown timer with game-over state
- Smooth animations and visual feedback on click
- Clean and modern UI built with Tailwind CSS

## 🛠 Tech Stack
- React 
- Vite
- JavaScript 
- Tailwind CSS
